# Открытка для Аружан

Готовый одностраничный сайт для публикации через GitHub Pages.

## 1. Добавь фотографии

Положи свои фотографии в папку `assets` и назови их строго так:

- `photo1.jpg`
- `photo2.jpg`
- `photo3.jpg`
- `photo4.jpg`
- `photo5.jpg`
- `gift.jpg`

Рекомендуемый формат фотографий: вертикальный, примерно 4:5.  
Например: 1200×1500 px.

## 2. Измени тексты

Открой файл `index.html` в любом редакторе.

Можно заменить:

- подписи под фотографиями;
- заголовки пожеланий;
- тексты пожеланий;
- текст подарка;
- подпись «С любовью, твой Алибек».

## 3. Проверь локально

Самый простой способ:

1. Открой папку проекта в Visual Studio Code.
2. Установи расширение `Live Server`.
3. Нажми правой кнопкой на `index.html`.
4. Выбери `Open with Live Server`.

Можно также просто открыть `index.html` двойным кликом, но Live Server надёжнее.

## 4. Опубликуй через GitHub Pages

1. Создай новый публичный репозиторий, например:
   `aruzhan-birthday`
2. Загрузи туда содержимое этой папки.
3. В GitHub открой:
   `Settings → Pages`
4. В разделе `Build and deployment` выбери:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
5. Нажми `Save`.

Через некоторое время сайт будет доступен по адресу:

`https://ТВОЙ-ЛОГИН.github.io/aruzhan-birthday/`

## Важно

Не меняй имена файлов `index.html`, `style.css`, `script.js`, если не меняешь ссылки на них внутри проекта.
